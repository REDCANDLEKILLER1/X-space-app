Same-Browser Setup:
1. Open this mixer in a Chrome tab
2. Start Engine → enable Monitor
3. Open X Spaces in a second Chrome tab
4. In X Spaces, go to Settings → Audio and select your output device
5. Android: Install AudioRelay or similar to route internal audio as virtual mic input → X picks up the full mix
6. Alternative: Use a 3.5mm loopback cable (headphone out → mic in) for hardware routing

Quick Mode: Enable Monitor, plug in earbuds, X Spaces mic will pick up room audio + speaker bleed. Works for casual streams.
